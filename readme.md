리얼 타임 과제
=====
과제 소개
---
웹소켓을 활용한 서버 실시간 소통 플랫포머 게임. 그런데 크롬 다이노를 곁드린.
서버 안에 assets을 사용해서 진행되고 어뷰징등 부적절한 사용 버그를 서버에서 관리함.

### 기획

1. 시간에 따라 점수가 증가
2. 점수에 따라 스테이지가 달라짐.
3. 스테이지가 올라갈수록 시간당 획득 점수가 늘어남.
4. 점수를 늘려주는 아이템이 등장하며 스테이지가 올라갈수록 더 많은 점수를 주는 아이템이 등장함.
